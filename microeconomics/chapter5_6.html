บทที่ 5.6: ต้นทุน 7 ประเภทในระยะสั้น

5.6เปิดเรื่อง

ต้นทุน 7 ประเภท — ฟังดูเยอะ แต่จริง ๆ มีแค่ 3 กลุ่ม

ใน 5.5 เราเรียน Explicit + Implicit = ต้นทุนเศรษฐศาสตร์
ทีนี้มาดูว่า ในระยะสั้น ต้นทุนทั้งหมดแบ่งย่อยตามพฤติกรรมได้ยังไงบ้าง

?

ถ้าร้านกาแฟขาย 0 แก้วทั้งเดือน
ต้นทุน = 0 บาท จริงหรือ?

ไม่ใช่! ต่อให้ไม่ขายเลย คุณยังต้องจ่าย:

🏠 ค่าเช่าตึก · 📄 ค่าประกัน · 🏦 ค่าผ่อนเครื่องชง (คงที่)

แต่สิ่งที่ไม่ต้องจ่าย: ค่าเมล็ดกาแฟ · ค่านม · ไม่ต้องจ้างคน (เพราะไม่มีอะไรให้ชง)

สังเกต: ต้นทุนแบ่งเป็น 2 กลุ่ม — พวกที่ยังคงจ่ายอยู่ กับพวกที่หายไปถ้าไม่ผลิต

นี่คือจุดเริ่มของ ต้นทุนคงที่ กับ ต้นทุนแปรผัน ที่เรากำลังจะไปเรียน:

🏠 ต้นทุนคงที่

  • จ่ายเท่าเดิมทุกเดือน
  • ไม่ขึ้นกับว่าขายได้กี่แก้ว
  • ผลิต 0 แก้วก็ยังจ่าย

ตัวอย่าง

ค่าเช่า ประกัน ค่าผ่อนเครื่อง

🥤 ต้นทุนแปรผัน

  • เปลี่ยนตามจำนวนที่ผลิต
  • ผลิตมาก = จ่ายมาก
  • ผลิต 0 แก้ว = จ่าย 0

ตัวอย่าง

เมล็ดกาแฟ นม ค่าจ้าง
จะได้รู้จัก 7 ตัว แบ่งเป็น 3 กลุ่ม

📦 Total (3 ตัว): TFC · TVC · TC — ต้นทุนรวม
📊 Average (3 ตัว): AFC · AVC · AC — ต้นทุนต่อแก้ว
➕ Marginal (1 ตัว): MC — ต้นทุนของแก้วใหม่

5.6กลุ่มที่ 1 · ต้นทุนรวม 3 ตัว

TFC, TVC, TC — 3 ตัวแรก ที่เห็นภาพง่ายที่สุด

ต้นทุนคงที่รวม

TFC

ต้นทุนทุกอย่างที่ไม่เปลี่ยนตามจำนวนที่ผลิต

ร้านคุณน้อย: ค่าเช่าตึก 20,000 + ค่าผ่อนเครื่องชง 10,000 = TFC = 30,000/เดือน

ต้นทุนแปรผันรวม

TVC

ต้นทุนทุกอย่างที่เปลี่ยนตามจำนวนที่ผลิต (Q)

ร้านคุณน้อย: ค่าวัตถุดิบ+ค่าจ้าง รวม 20 บาท/แก้ว
ถ้าขาย 1,000 แก้ว → TVC = 20 × 1,000 = 20,000

ต้นทุนรวมทั้งหมด

TC

= TFC + TVC · รวมทุกอย่างที่จ่าย

TC = TFC + TVC

ลองเล่น — เลื่อนจำนวนแก้วที่ขาย ดูต้นทุน 3 ตัว:

ลากแถบ — สังเกตว่า TFC ไม่เปลี่ยน แต่ TVC ขึ้นตาม Q
สรุป 3 ตัวแรก

🟠 TFC = คงที่เสมอ (เส้นแนวนอน)
🟢 TVC = เพิ่มตาม Q (เส้นขึ้น)
🟣 TC = TFC + TVC (แถบเต็ม)

5.6กลุ่มที่ 2 · ต้นทุนเฉลี่ย 3 ตัว

"ต่อแก้วเท่าไร?" — เอาต้นทุนรวม หารด้วย Q

Total ทำให้เราเห็นภาพรวม แต่จะตั้งราคาขายต่อแก้ว เราต้องรู้ต้นทุนเฉลี่ยต่อแก้ว → ได้อีก 3 ตัว:

ต้นทุนคงที่เฉลี่ย

AFC

TFC หารด้วย Q = ค่าคงที่เฉลี่ยต่อแก้ว

AFC = TFC ÷ Q
ตัวอย่างที่ Q = 1,000 แก้ว
TFC 30,000
Q 1,000
→
AFC
30 บาท/แก้ว

ข้อสังเกตสำคัญ: ยิ่ง Q เยอะ AFC ยิ่งน้อย (ต้นทุนคงที่ถูกเฉลี่ยกับแก้วเยอะขึ้น)

ต้นทุนแปรผันเฉลี่ย

AVC

TVC หารด้วย Q = ค่าแปรผันเฉลี่ยต่อแก้ว

AVC = TVC ÷ Q
ร้านคุณน้อย: วัตถุดิบ+ค่าจ้าง = 20 บาท/แก้ว → AVC = 20 (ค่อนข้างคงที่ในช่วงผลิตปกติ)

ต้นทุนเฉลี่ยรวม

AC

= AFC + AVC · ต้นทุนเฉลี่ยต่อแก้วรวมทุกอย่าง

AC = TC ÷ Q = AFC + AVC

เลื่อนดู — สังเกต AFC ลด แต่ AVC คงที่:

สรุป 3 ตัวเฉลี่ย

AFC = ลดลงเสมอเมื่อ Q เพิ่ม (ของคงที่กระจายไปแก้วเยอะขึ้น)
AVC ≈ คงที่ต่อแก้ว (ในช่วงผลิตปกติ)
AC = AFC + AVC

5.6กลุ่มที่ 3 · ต้นทุนของแก้วใหม่

MC — "แก้วถัดไป ต้องจ่ายเพิ่มเท่าไร?"

ยังจำ MP จาก 5.2 ได้ไหม? — "คนใหม่เพิ่มผลผลิตเท่าไร"
MC คือคู่แฝดทางฝั่งต้นทุน:

🔗 ความคล้ายคลึง

MP
ผลผลิตที่เพิ่ม
เมื่อเพิ่มคน 1 คน
⟷
MC
ต้นทุนที่เพิ่ม
เมื่อเพิ่มแก้ว 1 แก้ว

ต้นทุนเพิ่ม / ต้นทุนหน่วยสุดท้าย

MC

ต้นทุนที่เพิ่มขึ้นเมื่อผลิตแก้วใหม่ 1 แก้ว

MC = ΔTC ÷ ΔQ
ขาย 100 แก้ว → TC = 32,000 · ขาย 101 แก้ว → TC = 32,022
→ MC = 32,022 − 32,000 = 22 บาท (ต้นทุนของแก้วที่ 101)

สำคัญ: MC ใช้ตอบคำถาม "ควรผลิตเพิ่มไหม?" — เหมือน MP ตอบ "ควรจ้างคนเพิ่มไหม?"

🎯 เพราะ TFC ไม่เปลี่ยนเมื่อเพิ่ม Q

ΔTC = Δ(TFC + TVC) = ΔTFC + ΔTVC = 0 + ΔTVC = ΔTVC

ดังนั้น MC = ΔTVC ÷ ΔQ — ต้นทุนแปรผันเท่านั้นที่มีผลต่อ MC

💡 นัยสำคัญ: เวลาตัดสินใจผลิตแก้วถัดไป ต้นทุนคงที่ไม่เกี่ยว (จ่ายไปแล้วไม่ว่าจะผลิตหรือไม่) — ดูแค่ MC พอ

MC — ตัวที่ 7 ตัวสุดท้าย

💰 MC = ต้นทุนของแก้วถัดไป
🔗 คู่แฝดกับ MP (ฝั่งผลิต)
📌 MC มาจาก TVC เท่านั้น (ไม่เกี่ยว TFC)
