Introduction to Econometrics — Cheat Sheet

Ajarn Kwan
facebook.com/ajarnkwan

🤔 เศรษฐมิติคืออะไร?

ใช้ ข้อมูลจริง ทดสอบว่าทฤษฎีเศรษฐศาสตร์จริงไหม ไม่ใช่แค่เดา

ทฤษฎีเศรษฐศาสตร์ + คณิต + สถิติ + ข้อมูลจริง = เศรษฐมิติ ✨

📐 สมการพื้นฐาน

Y = a + bX Y = สิ่งที่อยากรู้  |  X = สิ่งที่มีผล  |  a = จุดเริ่มต้น  |  b = อัตราการเปลี่ยน
💡 b คือตัวสำคัญ: "ถ้า X เพิ่ม 1 → Y เปลี่ยนเท่าไหร่"
เช่น ไอศกรีม: ราคา = 10 + 30(สกูป) → สกูปละ 30 บาท

🏷️ ตัวแปรตาม vs ตัวแปรต้น

Y = ตัวแปรตาม (Dependent) — สิ่งที่อยากอธิบาย
X = ตัวแปรต้น (Independent) — สิ่งที่ใช้อธิบาย
💡 ตัวแปรตาม "ตาม" ตัวอื่น เช่น ยอดขายตามอุณหภูมิ

⚠️ Correlation ≠ Causation

"สัมพันธ์กัน" ≠ "เป็นสาเหตุ"

ยอดขายไอศกรีม ↑ พร้อมกับ คนจมน้ำ ↑ → ไม่ใช่ไอศกรีมทำให้จมน้ำ! แต่เพราะ "อากาศร้อน" (ตัวแปรซ่อน)

r = ค่าสหสัมพันธ์: +1 ไปด้วยกัน | 0 ไม่เกี่ยว | −1 สวนทาง

ระวัง Confounding Variable = ตัวแปรซ่อนที่มีผลต่อทั้ง X และ Y

📉 Regression (การถดถอยเชิงเส้น) — หัวใจของเศรษฐมิติ

Y = β₀ + β₁X + ε β₀ = จุดตัดแกน Y (Intercept)  |  β₁ = ความชัน (Slope)  |  ε = Error
β₁ = ตัวที่สำคัญที่สุด! "X เพิ่ม 1 หน่วย → Y เปลี่ยนเท่าไหร่ โดยเฉลี่ย"
ε = สิ่งที่โมเดลอธิบายไม่ได้ (โชค, ความสามารถ, ปัจจัยอื่นๆ)

R² (R-squared) = อธิบายได้กี่ %

0 = ไม่ได้เลย0.751 = สมบูรณ์

R² = 0.75 → โมเดลอธิบายการเปลี่ยนแปลงของ Y ได้ 75%
R² สูง ≠ ดีเสมอไป (ระวัง overfitting)

💡 เช่น เงินเดือน = 15,000 + 3,000(ปีที่เรียน) → เรียนเพิ่ม 1 ปี เงินเดือนเพิ่มเฉลี่ย 3,000 บาท | คนเรียน 16 ปี (ป.ตรี): 15,000 + 3,000×16 = 63,000 บาท

🗺️ แผนที่วิชา (จะเจอเมื่อเรียนจริง)

1 OLS — วิธีคำนวณหาเส้น regression ที่ดีที่สุด
2 Hypothesis Testing — ทดสอบว่า β₁ จริงๆ ≠ 0 ไหม
3 Multiple Regression — ใส่ตัวแปร X หลายตัว
4 ปัญหาที่ต้องระวัง — Multicollinearity, Heteroscedasticity
5 หัวข้อขั้นสูง — Time Series, Panel Data, IV
